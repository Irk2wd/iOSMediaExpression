# ImageProcessor

一款 iOS 端的媒体压缩工具，支持图片和视频的压缩处理。

## 功能

### 图片压缩
- 支持 **JPEG** 和 **PNG** 两种输出格式
- 自定义目标文件大小（KB）
- JPEG：通过二分查找自动调整压缩质量，必要时缩小尺寸
- PNG：通过缩小尺寸实现无损压缩
- 压缩后可直接保存到相册

### 视频压缩
- 输出格式为 **HEVC (H.265)**，压缩效率高
- 自定义目标文件大小（MB）
- 实时显示压缩进度
- 压缩后可直接保存到相册

## 技术栈

- SwiftUI
- PhotosUI / Photos
- AVFoundation / AVKit

## 系统要求

- iOS 16.0+
- 支持 HEVC 硬件编码的设备（iPhone 7 及以上）
